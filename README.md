# TrendAgent AI

An autonomous market intelligence platform that automates news research, performs sentiment analysis, and generates real-time market mood reports.

## Features
- Google News scraping
- TF-IDF keyword extraction
- Sentiment Analysis
- Market Mood Index
- Interactive Dashboard
- Markdown Report Export

## Tech Stack
- Python
- FastAPI
- JavaScript (ES6)
- HTML5/CSS3
- NLP
- TF-IDF
- XML Parsing
